# 비속어 필터 경고 메시지 수정 요약

## 문제점

1. **카운트 기능 사라짐**: 1회, 2회, 3회 카운트가 메시지에 명확히 표시되지 않음
2. **메시지 내용**: 현재 메시지가 너무 길거나 명확하지 않음
3. **요구사항**:
   - 1회, 2회: 해당 채팅 기록이 비속어 사용으로 저장되었다는 메시지 (간단하게)
   - 3회: 관리자에게 보고되었다는 메시지

## 수정 내용

### 경고 메시지 개선

**위치**: `server/bot/moderation/profanityFilter.js` (라인 244-268)

**변경 전**:
- 1회: "⚠️ [닉네임]님, 비속어 사용 시 강퇴될 수 있습니다."
- 2회: "⚠️ [닉네임]님, 비속어 사용 시 강퇴될 수 있습니다. (2회 경고)"
- 3회: "🚨 [닉네임]님, 운영진에게 보고됩니다. 강퇴 대상자 등록되었습니다. (3회 경고)"

**변경 후**:
- 1회: "⚠️ [닉네임]님, 비속어 사용이 감지되어 채팅 기록에 저장되었습니다."
- 2회: "⚠️ [닉네임]님, 비속어 사용이 감지되어 채팅 기록에 저장되었습니다. (2회)"
- 3회: "🚨 [닉네임]님, 비속어 사용이 관리자에게 보고되었습니다."

### 개선 사항

1. **간결한 메시지**: 불필요한 문구 제거, 핵심 내용만 전달
2. **명확한 카운트**: 2회 경고 시 "(2회)" 표시
3. **단계별 안내**:
   - 1-2회: 채팅 기록 저장 안내
   - 3회: 관리자 보고 안내
